Git Configuration Cheatsheet

Setting up GitHub E-Mail 

git config --global user.email "user@example.com"

Setting up Vim as Default Editor for Commits

git config --global core.editor "vim"

Show Name of Branch in CommandLine Prompt

Insert the following line to /home/user/.bashrc 
PS1='${debian_chroot:+($debian_chroot)}\u@\h:\w\[\033[0;32m\]$(__git_ps1)\[\033[0m\]\$ '

System Configuration Cheatsheet

Colorize Commandline Prompt for root

Insert the following line to /root/.bashrc 
PS1='${debian_chroot:+($debian_chroot)}\[\033[01;31m\]\u@\h\[\033[00m\]:\[\033[01;34m\]\w\[\033[00m\]\$ '
